durian.forms.create_match_forms(name, provides_args)¶

With a list of supported arguments, generate a list of match forms for these.

E.g. if the supported arguments is ["name", "address"], it will generate forms like these:

Name: SELECT:[any|exact|starts with|ends with|contains] [ query ]
Address: SELECT:[any|exact|starts with|ends with|contains] [ query ]

When these form are feeded with data they can be converted to a match dict like:

>>> {"name": Startswith("foo"), "address": Endswith("New York")}
